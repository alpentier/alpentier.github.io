David Sampson > Work > CUED Technical Report

D. J. M. Sampson, Active Roll Control of Articulated Heavy Vehicles, Technical Report CUED/C-MECH/TR 82, Department of Engineering, University of Cambridge, UK, 2002.

This report is based on work originally published in my Ph.D. dissertation of the same name. The key difference between the dissertation and this report is that the tyre parameters in this report have been updated to better represent average truck tyres, and in particular, the tyres to be used on the Cambridge Vehicle Dynamics Consortium's experimental vehicle. All controllers have been redesigned and all results recalculated for these amended tyre parameters.

The full report is available online (PDF, 3482 KB).
